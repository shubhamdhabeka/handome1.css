What is CSS?

  1. CSS stands for cascading style sheets.

  2. CSS describe how HTML elements are to be displayedon screen,paper,or in other media.

  3. CSS is not neccessory css saves a lot of work. can control the layout of multiple web pages all at one

  4. stylesheet are stored in CSS files.

CSS can be added to HTML document in 3 ways:

  • inline-by using the style attribute inside HTML elements
  • Internal-by using style elements in the head section
  • External-by using a link element to link to an external css file.
  • the most common way to add css,is the keep the style in external css files.however,in this tutorial we will use inline and internal styles, because this is easier to demonstrate,and easier for you to try it yourself

when compairing css class vs ID,the difference is that css class applies astyle to multiple elements.id,on the other hand,applies a special in that you can use a soecial url to linl directly to an elements and its use by javascript.

in css,selectors are use to target a specific elements or a range of elements on a web page.once an element has been targeted,a style or set of style can be applied to the elements

there is a wide range of selectors available.two of the most commonly used are class and id.both are used to target elements to which style should be applied
